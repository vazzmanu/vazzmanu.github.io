Tic Tac Toe

18/03/21

A simple but challenging enough program you can make while learning to code python is a Tic Tac Toe, but out there exist two types of programs that accomplish this:

  • The beautiful and simple ones.
  • The ugly ones.

By keeping in mind some principles like:

  • KISS (Keep it simple)
  • DRY (Don't repeat yourself)

You can code something simple and short enough that everybody can underrstand and appreciate your brilliance.

If you want to see an example please check out my repo on Github.
